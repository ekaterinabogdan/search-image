# Image–Text Retrieval System

Проект по поиску изображений по текстовому описанию с использованием
BERT для текста и нейронной сети для оценки сходства «текст–картинка».

## Описание проекта
В работе реализован сервис, который:
- принимает текстовый запрос,
- преобразует его в векторное представление (BERT),
- сравнивает с векторами изображений,
- возвращает наиболее релевантные изображения.

## Используемые модели
- BERT (эмбеддинги текста)
- Полносвязная нейронная сеть для оценки сходства

## Метрики качества
Оценка проводилась на валидационной выборке с использованием:
- MSE
- MAE
- R²

Лучшая модель — нейронная сеть.
